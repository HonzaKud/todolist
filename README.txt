To-Do List – Vanilla JS + Node.js
Tento projekt je jednoduchá To-Do aplikace vytvořená jako osobní cvičení pro osvojení webových technologií – práce s frontendem ve vanilla JavaScriptu a backendem postaveným na Node.js a Expressu. Aplikace umožňuje uživateli přidávat úkoly, mazat je a označovat je jako dokončené. Data se ukládají do MongoDB databáze a zůstávají zachována i po obnovení stránky.
Cílem projektu bylo propojit funkční backend s jednoduchým frontendem a pochopit, jak funguje asynchronní komunikace mezi klientem a serverem.
________________________________________
Použité technologie
•	HTML, CSS, JavaScript (vanilla JS) – základní frontend, bez použití frameworků
•	Node.js + Express.js – backend pro zpracování API požadavků
•	MongoDB + Mongoose – databáze a modelování dat
•	Render – nasazení backendu
•	GitHub Pages – hostování frontendové části
________________________________________
Funkce
•	Přidávání nového úkolu pomocí formuláře
•	Zobrazení seznamu úkolů (nedokončené a dokončené)
•	Označení úkolu jako hotového nebo jeho vrácení mezi nedokončené
•	Smazání úkolu
•	Všechna data se ukládají do MongoDB
________________________________________
Co zatím není hotové
•	Editace úkolu
•	Přihlášení/registrace uživatele – aplikace momentálně nemá žádný autentizační systém
•	Responsivita – základní responzivita řešena přes Bootstrap
•	React – frontend zatím nevyužívá žádný framework, přepis do Reactu je v plánu
________________________________________
Cíl projektu
Tento projekt vznikl jako praktické cvičení pro propojení frontendové a backendové logiky. Šlo mi o to vyzkoušet komunikaci s API, práci s databází MongoDB a základy nasazování projektů online.
